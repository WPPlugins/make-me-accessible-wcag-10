== Description ==

With this tool you can add many new features to your website to make it accessible like the W3C'view of accessibility (http://www.w3.org/TR/WCAG10/). 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `makeMeAccessible.php` and the folder `makeMeAccessible` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
